# Context Fusion

Run the Context Fusion skill for `$ARGUMENTS`.

1. Read the Context Fusion skill at `skills/context-fusion/SKILL.md` and follow its instructions exactly.
2. Locate all pipeline artifacts for this topic — research, PRDs (all versions), design specs, prototypes, debate transcripts, evaluations.
3. Produce a unified context document that reconciles all artifacts: what's consistent across sources, what conflicts, what's missing, what's evolved.
4. Flag any contradictions between artifacts (e.g., PRD says X but prototype implements Y).
5. Output: context fusion document saved to the pipeline working directory.
